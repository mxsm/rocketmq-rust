# Ballpit 动态背景示例

这是一个使用 Ballpit 动态背景的示例页面。

## 特性

- ✨ 动态球体背景动画
- 🖱️ 鼠标交互效果（鼠标靠近球体时会变大）
- 📱 移动端自适应（默认禁用）
- 🎨 可自定义颜色和配置
- ⚡ 性能优化

## 如何使用

1. 在页面 front matter 中添加 `ballpit_bg: true`
2. 设置 `layout: splash-ballpit`
3. 配置其他参数（可选）
