# 关于作者

## MXSM

MXSM 是 RocketMQ-Rust 的创建者和维护者。

### 背景

- **专长**：分布式系统、消息中间件、Rust 编程
- **重点**：构建生产就绪、内存安全的 Apache RocketMQ 实现
- **理念**：利用 Rust 的所有权模型和异步能力实现企业级可靠性

### 贡献

MXSM 积极开发和维护 RocketMQ-Rust，提供：

- 利用零成本抽象的高性能消息处理
- 通过 Rust 的类型系统保证内存安全
- 对 Linux、Windows 和 macOS 的跨平台支持
- 面向现代分布式系统的生产就绪功能
