# 安装 RocketMQ-Rust

## 环境要求

- Rust 1.70 或更高版本
- Cargo 包管理器

## 从源码构建

```bash
git clone https://github.com/mxsm/rocketmq-rust.git
cd rocketmq-rust
cargo build --release
```

## 验证安装

```bash
rocketmq-rust --version
```
